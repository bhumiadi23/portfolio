# Personal Developer Portfolio

Welcome to my personal developer portfolio! This interactive website showcases my skills, experience, and the various fullstack projects I've built. 

## 🚀 Features
- **Modern UI/UX**: Built with React and designed for a smooth, responsive user experience.
- **Interactive Elements**: Features dynamic components like an integrated AI chatbot (AdiBot) and a sleek Hero Section.
- **Project Showcase**: Highlights my journey and full-stack projects including web applications, AI tools, and enthusiast platforms.

## 🛠️ Tech Stack
- Frontend: React, Vite, CSS
- Deployment: GitHub Pages / Vercel (Ready)

## 🏃‍♂️ Running Locally
1. Clone the repository
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`
