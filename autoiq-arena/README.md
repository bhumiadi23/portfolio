## 🌟 Overview

**AutoIQ Arena** redefines the way automotive enthusiasts interact with car data. Emphasizing a sleek, premium dark-themed aesthetic with modern glassmorphism UI elements, this platform offers an immersive, interactive space to explore, simulate, and compare the world's most impressive vehicles. 

Whether you're comparing 0-100 km/h acceleration graphs or taking advice from the integrated AI Advisor, AutoIQ Arena serves as your ultimate digital garage.

## ✨ Core Features

- 🏁 **Interactive Dashboard:** Browse curated collections of luxury, sports, and EV vehicles with gorgeous hover states and transitions.
- 🚦 **Performance Simulator:** Real-time drag race visualizers, dynamic speedometer gauges, and acceleration charts powered by `recharts`.
- 🤖 **AI Auto Advisor:** Get intelligent recommendations and insights for your next vehicle purchase or modification.
- ⚡ **EV Tools Hub:** Dedicated calculators for electric vehicle range, charging times, and cost-benefit analysis.
- 👥 **Community Integration:** Discuss builds, share garage collections, and join an enthusiastic automotive community.
- 🎨 **Premium Aesthetics:** Stunning dark-mode UI, neon accenting, and buttery smooth animations driven by Framer Motion.

## 🛠️ Tech Stack Architecture

- **Frontend Core:** React 18, TypeScript, Vite
- **Styling & UI:** Tailwind CSS, Glassmorphism utility patterns
- **Animations:** Framer Motion (page transitions, micro-interactions)
- **Data Visualization:** Recharts (performance graphs, torque/HP curves)
- **Routing:** React Router v6

## 🚀 Getting Started

To run AutoIQ Arena locally on your machine, follow these steps:

### Prerequisites
- Node.js (v16.0 or higher)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/bhumiadi23/autoiq-arena.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd autoiq-arena
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Start the development server:**
   ```bash
   npm run dev
   ```
